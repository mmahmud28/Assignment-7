KeenKeeper (Friendship Tracker App)

📝 Short Description

KeenKeeper is a personal relationship management web app that helps users track, organize, and maintain meaningful connections with friends. It allows users to log interactions like calls, texts, and video chats, and visualize relationship activity through analytics.

⚙️ Technologies Used
Next.js (App Router)
React.js
JavaScript (ES6+)
Tailwind CSS
LocalStorage API
Recharts (for analytics)
React Icons

1. 📇 Friend Management System

Users can add, view, and manage friends with profile details, tags, and status indicators.

2. 📊 Interaction Tracking & Analytics

Track interactions (Call, Text, Video) and visualize them using interactive pie charts.

3. ⏰ Relationship Status Monitoring

Automatically shows relationship status like On Track, Almost Due, Overdue based on last contact days.
